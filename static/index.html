Getting Started

Welcome to Contact Finder Pro! Here's how to get started:

  1. Go to Settings and add your Perplexity API key
  2. Return to Search and enter what you're looking for
  3. Click Search and view your results
  4. Export results in your preferred format

Search Tips

Effective Search Queries

Good Examples:

  • "roofing contractors in Austin Texas owner contact"
  • "restaurant owners downtown Chicago email list"
  • "medical practices Houston decision makers"

Search Modes

  • Standard: Faster, returns 5-10 results
  • Enhanced: Slower, returns 20+ results with deeper searching

Understanding Results

  • Name: Contact person's name
  • Company: Business name
  • Email/Phone: Contact information found
  • Confidence: How confident we are in the accuracy
  • Sources: Number of sources that confirm this information
  • Verification: Whether email/phone has been verified

Export Formats

  • CSV: Opens in Excel, Google Sheets
  • Excel: Full spreadsheet with multiple sheets
  • JSON: For developers and advanced users
  • Apollo: Compatible with Apollo.io CRM
